What is Bootstrap?

Bootstrap is the world's most popular open-source CSS framework. It provides pre-styled components, a powerful responsive grid system, and utility classes that let developers build consistent, mobile-first websites quickly.

Including Bootstrap via CDN

index.html 
<!-- Bootstrap CSS — add inside <head> -->
<link rel="stylesheet"
  href="https://cdn.jsdelivr.net/npm/bootstrap@5.3/dist/css/bootstrap.min.css">

<!-- Bootstrap JS — add before </body> -->
<script
  src="https://cdn.jsdelivr.net/npm/bootstrap@5.3/dist/js/bootstrap.bundle.min.js"
></script>
CDN: Including Bootstrap from a CDN means you don't need to download and host the files yourself. Since many websites use Bootstrap, it's often already cached in the user's browser — making your page load faster.

The Bootstrap Grid System

Bootstrap uses a 12-column grid. You specify how many columns each element occupies at each screen size using classes like col-md-4 (4 out of 12 = one third).

grid.html 
<div class="container">
  <div class="row">
    <div class="col-12 col-md-4">Column 1</div>
    <div class="col-12 col-md-4">Column 2</div>
    <div class="col-12 col-md-4">Column 3</div>
  </div>
</div>
Responsive breakpoints: col-12 col-md-4 means: full width on small screens, one third on medium screens and above. This is Bootstrap's mobile-first responsive design — one set of classes handles all screen sizes.
