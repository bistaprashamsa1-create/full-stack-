What is CSS?

CSS (Cascading Style Sheets) controls the visual presentation of HTML elements — colours, fonts, spacing, layout, and animations. CSS separates content (HTML) from presentation, making code cleaner and easier to maintain.

The Box Model

Every HTML element is a rectangular box with: content, padding, border, and margin.

boxmodel.css 
.card {
  width:         300px;   /* content width */
  padding:       20px;    /* space inside border */
  border:        2px solid #2c3e5a;
  margin:        16px;    /* space outside border */
  border-radius: 8px;     /* rounded corners */
}

grid.css 
.grid-container {
  display: grid;
  grid-template-columns: repeat(3, 1fr);
  gap: 0.75rem;
}

.grid-box:hover {
  transform: translateY(-4px);
  box-shadow: 0 8px 20px rgba(0,0,0,0.2);
  transition: all 0.25s ease;
}
This demonstrates: display: grid with repeat(3, 1fr) — the 1fr unit means each column takes one equal fraction of the available space. The hover effect uses transform and transition for smooth animation — no JavaScript required.

Live Demo — CSS Animations

CSS @keyframes define how an element changes over time. The animation property applies it to an element.

Pure CSS — no JavaScript

bounce
spin
pulse
fade
animations.css 
@keyframes bounce {
  0%, 100% { transform: translateY(0); }
  50%       { transform: translateY(-28px); }
}

@keyframes spin {
  to { transform: rotate(360deg); }
}

.ball {
  animation: bounce 1.4s infinite ease-in-out;
}
This demonstrates: The @keyframes rule defines animation states at percentage breakpoints. The animation shorthand sets the name, duration, iteration count, and timing function. All four animations above are CSS-only.

CSS Custom Properties (Variables)

CSS variables let you define reusable values once and reference them throughout your stylesheet.

variables.css 
:root {
  --primary-color:  #2c3e5a;
  --accent-color:   #2980b9;
  --font-size-base: 1rem;
  --border-radius:  8px;
}

.card {
  background:    var(--primary-color);
  border-radius: var(--border-radius);
  font-size:     var(--font-size-base);
}
Why use CSS variables? If you need to change your brand colour, you update one line in :root instead of finding and replacing it across hundreds of CSS rules.
