What is HTML?

HTML (HyperText Markup Language) is the standard language for creating web pages. It describes the structure of a page using elements represented by tags. Every web page you visit is built on HTML.

Semantic HTML

Semantic HTML uses meaningful element names that describe their purpose. This improves accessibility, SEO, and code readability.

structure.html 
<!-- Semantic HTML uses meaningful tag names -->
<header>
  <nav>
    <a href="/">Home</a>
    <a href="/about">About</a>
  </nav>
</header>

<main>
  <article>
    <h1>Page Heading</h1>
    <section>
      <p>Content goes here...</p>
    </section>
  </article>
  <aside>Sidebar / Related links</aside>
</main>

<footer>
  <p>© 2025 Prashamsha Khadka</p>
</footer>
Why it matters: Tags like <header>, <main>, and <footer> help screen readers understand the page (accessibility), help search engines index content better (SEO), and make the code easier to maintain.
