What is JavaScript?

JavaScript is the programming language of the web. While HTML provides structure and CSS provides style, JavaScript adds behaviour and interactivity. It can respond to user actions, update content without reloading, validate forms, and communicate with servers.

JavaScript runs in the browser (client-side) but can also run on servers using Node.js, making it a true full stack language.

counter.js 
let count = 0;

function changeCount(direction) {
  if (direction === 0) {
    count = 0;
  } else {
    count += direction;
  }
  // DOM manipulation — update the element's text
  const display = document.getElementById('counterDisplay');
  display.textContent = count;
}
This demonstrates: A JavaScript variable (let count) holds state. The function modifies it, then uses document.getElementById() to select the element and updates its textContent — this is called DOM manipulation.

function addTodo() {
  const input = document.getElementById('todoInput');
  const text  = input.value.trim();
  if (!text) return;

  const li = document.createElement('li');
  li.className = 'todo-item';
  li.innerHTML = `
    <div class="todo-check" onclick="toggleTodo(this)">✓</div>
    <span class="todo-text">${text}</span>
    <button class="todo-del" onclick="deleteTodo(this)">✕</button>
  `;
  document.getElementById('todoList').prepend(li);
  input.value = '';
  input.focus();
}

function toggleTodo(btn) {
  btn.closest('.todo-item').classList.toggle('done');
  btn.classList.toggle('checked');
}

function deleteTodo(btn) {
  btn.closest('.todo-item').remove();
}
This demonstrates: Dynamic DOM creation with createElement() and innerHTML. classList.toggle() switches CSS classes to change visual state. .closest() traverses the DOM tree to find a parent element. onkeydown detects when Enter is pressed.
